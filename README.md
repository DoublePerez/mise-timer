# Pomodoro

A focus timer designed with editorial precision. Dieter Rams principles applied to a web app. Every element has earned its place.

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- Geist + Geist Mono (Vercel)

## Run

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Structure

```
src/
  components/     # timer-display, mode-indicator, controls, time-selector
  hooks/          # use-pomodoro
  lib/            # types, constants, utils
```

## Stages

- **Stage 1** — Core timer. Done.
- **Stage 2** — Personalization + feedback.
- **Stage 3** — Polish + PWA.
